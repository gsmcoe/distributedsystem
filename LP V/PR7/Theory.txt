Assignment No. 7: Distributed Application to Consume the Web Service

Aim: To create a simple web service and write any distributed application to consume the web service.

Web service is a software system used for communication between different applications over a network. It allows programs developed in different languages and running on different platforms to exchange information using standard protocols like HTTP, XML, SOAP, and REST. Web services are very important in distributed systems because they help different systems work together smoothly.

There are mainly two types of web services: SOAP and REST. SOAP stands for Simple Object Access Protocol. It is XML-based and provides secure communication between applications. REST stands for Representational State Transfer. It is simpler, faster, and uses HTTP methods like GET and POST for communication. REST is widely used because of its simplicity and better performance.

In web service architecture, there are three main roles: Service Provider, Service Requestor, and Service Registry. The Service Provider creates and hosts the service. The Service Requestor is the client that uses the service. The Service Registry stores information about available services so clients can find them easily.

In this practical, a simple calculator-based distributed application is created using Java Servlet. The project contains a class named CalculatorServlet.java which performs arithmetic operations like addition, subtraction, multiplication, and division.
The servlet receives two numbers from the user using input fields named txtN1 and txtN2. It also receives the selected operator using the parameter opr. Inside the doPost() method, the servlet reads these values using request.getParameter() and converts them into double values.

This practical helps in understanding how web applications communicate using request-response architecture. It also explains how server-side processing works using servlets in Java.

In conclusion, this assignment successfully demonstrates the creation of a simple web service using Java Servlet and a calculator application to consume it. It shows how distributed applications exchange data over a network and provide services to users. Web services are an important part of distributed systems because they improve communication, integration, and interoperability between applications.
